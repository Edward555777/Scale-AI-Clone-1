Quick Installation

Run Locally
# Clone repository
git clone https://github.com/Edward555777/Scale-AI-Clone-1.git

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
